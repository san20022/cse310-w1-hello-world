# Hello World - CSE 310

This project is a simple Python program that interacts with the user by asking for their name and welcoming them to the CSE 310 class. It demonstrates basic Python concepts like input, printing, string formatting, and simple loops.

## Instructions for Build and Use

[Software Demo](https://www.kaltura.com//p/1157612/embedPlaykitJs/uiconf_id/56813562?iframeembed=true&entry_id=1_lz8hna9b)

Steps to build and/or run the software:

1. Make sure Python 3.x is installed on your system.
2. Download the hello.py file from this repository.
3. Open a terminal or command prompt, navigate to the folder containing the file, and run:
   
bash
   python hello.py
